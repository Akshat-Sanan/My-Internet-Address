This game incorporates a local storage feature, enabling users to select correct letters from a list to match those in a randomly generated word, while also tracking and storing their highest score. Demonstrating fundamental JavaScript principles, each letter in the generated word appears precisely four times, diminishing incrementally in frequency, culminating with the final letter appearing only once in the grid.
